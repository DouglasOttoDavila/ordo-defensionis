Photo galleries for each asset live here.

Default convention per asset:

- `assets/gallery/<asset-slug>/01-overview.webp`
- `assets/gallery/<asset-slug>/02-detail.webp`
- `assets/gallery/<asset-slug>/03-deployment.webp`

Alternative extensions `.jpg` and `.png` are also checked automatically.

If you want custom titles, captions, credits, or non-standard file names, register the asset in:

- `src/config/assetMedia.ts`
