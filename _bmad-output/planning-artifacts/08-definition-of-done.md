# Definition of Done

## Phase 1 is complete only if all conditions below are met

### Research
- source inventory exists
- official/public source hierarchy is documented
- legal and reuse notes are documented
- key gaps and limits are documented
- LAI request templates are prepared

### Planning
- architecture is documented
- schema is documented
- provenance framework is documented
- confidence model is documented
- test strategy is documented
- implementation backlog exists

### Implementation
- repository is structured
- dependencies and environment are set up
- database exists with migrations
- at least one meaningful source ingests end-to-end
- evidence is stored
- confidence is stored
- conflicts are preserved
- tests pass
- documentation allows reproduction

### Quality
- no fabricated data
- no undocumented assumptions treated as facts
- no silent overwriting of contradictory claims
- source provenance is queryable
- reruns are reasonably safe

## Phase 2 may only start after user acceptance of Phase 1
Allowed next steps after approval:
- API
- dashboard
- search
- evidence UI
- exports
- analytics
